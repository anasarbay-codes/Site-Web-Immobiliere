# Maysanne Building - Site Web Immobilière

## Description du Projet

Ce projet est un site web statique pour Maysanne Building, une entreprise de construction immobilière marocaine spécialisée dans les projets résidentiels et tertiaires. Le site présente l'excellence de l'entreprise, ses réalisations et ses projets en cours.

## Fonctionnalités

- **Page d'accueil** : Présentation de l'entreprise avec vidéo de fond, métriques clés et navigation vers les sections principales
- **Projets** : Affichage des projets immobiliers avec liens vers les détails
- **Page "Coming Soon"** : Compte à rebours pour les projets à venir
- **Page de détails de projet** : Informations détaillées sur un projet spécifique (Maysanne Bouskoura)
- **Design responsive** : Adapté aux mobiles et tablettes
- **Navigation fluide** : Menu burger pour mobile

## Technologies Utilisées

- **HTML5** : Structure des pages
- **CSS3** : Styles et animations (fichier `immo.css`)
- **JavaScript** : Interactions et fonctionnalités (fichier `immo.js`)
- **Google Fonts** : Polices Playfair Display et Poppins
